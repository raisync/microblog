## Setup Laravel Website
<p>
	<ul>
		<li>git clone https://github.com/raisync/microblog_laravel-ionic.git</li>
		<li>composer install</li>
		<li>setup .env</li>
		<li>php artisan migrate</li>
	</ul>
</p>

## Setup Ionic Mobile App
<p>
	<ul>
		<li>cd mobile_app</li>
		<li>npm install</li>
		ionic serve
	</ul>
</p>
